Ce projet consiste en une page web simple qui permet aux utilisateurs de convertir des valeurs en euros en francs français. 
L'interface utilisateur est composée d'un champ de texte où les utilisateurs peuvent saisir une valeur en euros. 
La page est stylisée avec CSS pour afficher un arrière-plan, organiser les éléments de manière esthétique et utiliser une police personnalisée. 
Lorsque l'utilisateur saisit une valeur, le script JavaScript associé valide cette entrée à l'aide d'une expression régulière pour s'assurer qu'elle est numérique (acceptant les nombres entiers et à virgule). 
Si l'entrée est valide, elle est convertie en francs français en utilisant un taux de conversion fixe, et le résultat est affiché sur la page. 
Si l'entrée est invalide, un message d'erreur s'affiche, invitant l'utilisateur à saisir une valeur numérique valide.
